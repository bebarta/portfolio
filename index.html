Projects

2023

E-commerce Website - Toy Haven

In my first year as a web developer, I worked on an e-commerce website called Toy Haven. The project involved creating a user-friendly interface for browsing and purchasing toys online. I utilized HTML, CSS, and JavaScript to design the front end, ensuring responsive and attractive layouts for both desktop and mobile users. The back end was powered by Node.js and Express, with MongoDB handling the database. I also integrated payment gateways for seamless transactions.

2022

Blogging Platform - WriteAway

In my second year, I contributed to the development of a blogging platform called WriteAway. I was responsible for implementing the authentication system using JWT and securing the application against common web vulnerabilities. The project was built on the MERN stack (MongoDB, Express, React, Node.js), and I gained experience in state management with Redux. Additionally, I integrated a rich text editor for users to create engaging content.

2021

Project Management Dashboard - TaskMaster

During my third year, I collaborated on a project management dashboard named TaskMaster. This project aimed to streamline task tracking and team collaboration. I worked on the real-time features using Socket.io for instant updates. The front end was built with React, and the back end was a combination of Node.js and Express with a PostgreSQL database. I gained exposure to agile development methodologies and participated in daily stand-ups and sprint planning.

2020

Responsive Portfolio Website - Personal Showcase

Alongside these major projects, I also developed and maintained my personal portfolio website. This project allowed me to experiment with the latest technologies, implement responsive design principles, and showcase my skills and projects. I used Gatsby.js to optimize performance and deployed the site on Netlify, incorporating continuous integration for seamless updates.
